# Hermes Opportunity Analyzer – Agent Flow

## Step 1 – User Input
User defines a niche or problem area.

## Step 2 – Reddit Search Tool
Agent searches relevant subreddits using keyword expansion.

## Step 3 – Problem Clustering
Posts are grouped by semantic similarity.

## Step 4 – Payment Intent Detection
Signals detected:
- “I would pay for this”
- “Is there a tool for this?”
- “This is driving me crazy”

## Step 5 – Scoring Engine
Each cluster scored 0–100 based on:
- Frequency
- Engagement
- Frustration level
- Payment intent strength

## Step 6 – Opportunity Report
Structured JSON output generated.
